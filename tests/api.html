class Emitter

View source

Extends Object

This class can be used directly whenever an event emitter is needed. local emitter = Emitter:new() emitter:on('foo', p) emitter:emit('foo', 1, 2, 3) Also it can easily be sub-classed. local Custom = Emitter:extend() local c = Custom:new() c:on('bar', onBar)

function Emitter:wrap(name)

View source

Utility that binds the named method `self[name]` for use as a callback. The first argument (`err`) is re-routed to the "error" event instead. local Joystick = Emitter:extend() function Joystick:initialize(device) self:wrap("onOpen") FS.open(device, self.onOpen) end function Joystick:onOpen(fd) -- and so forth end

function Emitter:removeListener(name, callback)

View source

Remove a listener so that it no longer catches events.

function Emitter:once(name, callback)

View source

Same as `Emitter:on` except it de-registers itself after the first event.
